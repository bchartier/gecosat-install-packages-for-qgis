# Gestionnaire FT - Extension pour QGIS

Utilitaire pour installer des modules tiers dans la distribution Python accompagnant QGIS.
Ne fonctionne que sous Windows.

Modules actuellement installés (voir ) :

* SQLAlchemy
* pandas
* openpyxl
